IZSM-Esiti-Molluschi

Tramite questo cavaliere e' possibile inviare i campioni sui molluschi bivalvi in BDN.

Il file da caricare dovrà essere in formato XLS o XLSX e dovra' contenere, in quest'ordine, i seguenti campi (tutti obbligatori):
  • pianoCodice: campo di testo
  • PRELIEVI.numeroSchedaPrelievo: campo di testo
  • DATAPREL: campo di testo
  • luogoPrelievoCodice: campo di testo
  • metodoCampionamentoCodice: campo di testo
  • motivoCodice: campo di testo
  • prelNome: campo di testo
  • PrelCognome: campo di testo
  • prelCodFiscale: campo di testo
  • sitoCodice: campo di testo
  • comuneCodiceIstatParziale: campo di testo
  • siglaProvincia: campo di testo
  • laboratorioCodice: campo di testo
  • latitudine: campo di testo
  • longitudine: campo di testo
  • PRELIEVI.codiceContaminante: campo di testo
  • CAMPIONI MOLLUSCHI.numeroSchedaPrelievo: campo di testo
  • progressivoCampione : campo di testo
  • foodexCodice: campo di testo
  • CAMPIONI MOLLUSCHI.ProfFondale.ProfFondale: campo di testo
  • classificazioneDellaZonaDiMareCe8542004: campo di testo
Il ruolo abilitato all'utilizzo del cavaliere sarà ORSA. Caricato il file, il sistema:
  • provvedera' ad elaborarlo
  • ricerchera' eventuali errori
  • in assenza di errori effettuera' l'invio tramite WS alla BDN
  • produrra' un log per ogni invio effettuato con gli esiti OK e KO
