Reperibilita Asl

E' possibile visualizzare per ogni asl l'elenco delle reperibilita'. La visualizzazione di tali elenchi è disponibile a partire dal cavaliere 'Bacheca', selezionando il link "Elenco reperibilita' Asl" .






Ogni utente potra' visualizzare la lista (in effetti sara' possibile visualizzare i file inseriti dalle Asl con tutte le informazioni utili) e le reperibilita' di tutte le Asl pur non facendone parte. Cliccando sull'Asl interessata, l'utente autorizzato potra' aggiungere o cancellare i file presenti.
Cliccando sul link Allega lista Reperibilita'



e selezionato il file da caricare, inserendo il testo per loggetto ..





      .. si avra' la pubblicazione delle informazioni volute.
